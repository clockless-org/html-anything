html-anything

Turn any file into a beautiful, interactive, shareable HTML.

WhatsApp exports, PDFs, transcripts, CSVs, codebases — drop in a file, get back a single self-contained HTML with a modern reading experience: dark mode, search, filters, sortable tables. Open it. Send it. Host it. Done.

$ npx html-anything chat.zip
→ chat.html  (1 file, opens in any browser)
