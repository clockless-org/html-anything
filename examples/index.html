html-anything

Turn an idea, file, folder, or URL into a polished live HTML page — designed automatically for the source.

Give the skill a prompt or export. It figures out the source, chooses the page style, builds the interaction, and returns a shareable .html page.

Use html-anything to create an interactive teaching site about the solar system
→ output.html  (opens in any browser)

🧭Auto style systems

Timeline Story orders · media · reading · health · AI chats

Personal histories become scrollytelling pages with a time lens, chapter panels, rhythm strips, and a searchable memory drawer.

Map Atlas places · trips · routes · rideshare · photo metadata

Spatial sources open on an offline SVG/canvas stage with place drawers, period filters, route layers, and linked waypoint browsers.

Network Map contacts · email · Slack · Discord · LinkedIn · payments

People-shaped data becomes a graph, cluster map, or matrix with entity inspectors, bridge cards, privacy masks, and linked records.

Ops Console finance · logs · queues · spreadsheets · incidents

Dashboard layout is reserved for operational work: dense filters, KPI rails, anomaly queues, searchable tables, and copyable handoffs.
